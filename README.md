# Digit Analyser v2
## Description
This is a school project for learning how to program neural networks. It builds a neural network from scratch while also learning how to work with web technology through pywebview and building a production grade repository with good dx, documentation, buildscripts and a License. The neural network identifies hand drawn numbers and returns the probability of each possible number.

![UI Image](assets/UI_example.png)

## Installation
### Pre-Packaged .exe
You can just download the .exe file from the newest release.
### Package .exe from Source
You have to download the source files and open them ideally in PyCharm.
When using PyCharm you only have to run the already existing run config "Package App \[dist\]" and the packaged .exe file will be available in the __dist__ folder.
Alternatively you can just run the batch file named "build_app.bat" and it will do the same as the mentioned method above.
